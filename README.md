# Chit Chat

This is a simple chat application built using Node.js, Express, and MongoDB. It allows users to send and receive messages in real-time.

## How To Run App

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm run dev
```

## Notes

### MONGODB NOTES
- For a live application, we should specifically add the static IP addresses of our web server or hosting service in `IP Access List`
