# @chit-chat/typescript-config

Shared TypeScript configuration for the chit-chat monorepo.

## Usage

### For React applications (Vite):

```json
{
  "extends": "@chit-chat/typescript-config/react.json"
}
```

### For Node.js applications:

```json
{
  "extends": "@chit-chat/typescript-config/node.json"
}
```

### For other projects:

```json
{
  "extends": "@chit-chat/typescript-config/base.json"
}
```
